You will need to install [ImageMagick](http://www.imagemagick.org/script/index.php), [Node.js](https://nodejs.org/en/)

To run samples, hit:
```
git clone https://github.com/ilyaivanov/ResponsiveImages.git
cd ResponsiveImages
npm install gulp -g
npm install
gulp
```
